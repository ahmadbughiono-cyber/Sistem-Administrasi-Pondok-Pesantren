Sinkronisasi Cloud

Akses Masuk Aplikasi

Ganti kredensial masuk default Anda demi keamanan administrasi Pondok Pesantren Al Azhar secara berkala.

Panduan Integrasi Database

Ikuti langkah mudah berikut untuk menghubungkan sistem dengan Google Sheets:

  1. Buka Google Sheets > Klik Ekstensi > Pilih Apps Script.
  2. Tempel seluruh kode code.gs yang disediakan.
  3. Klik **Terapkan (Deploy)** sebagai Web App.
  4. Atur akses ke **"Siapa saja (Anyone)"** lalu salin URL yang diperoleh.
  5. Simpan URL di kolom atas. Status akan berubah menjadi **Tersambung (Cloud)**.
